Политика конфиденциальности

Приложения «Хатсайр» и «Хатсайр Водитель» уважают конфиденциальность пользователей и стремятся обеспечить защиту персональных данных.

1. Сбор информации

Приложение «Хатсайр» не собирает персональные данные пользователей.

Приложение «Хатсайр Водитель» может собирать данные о местоположении устройства для передачи текущей позиции маршрутки.

2. Использование данных

Данные о местоположении используются исключительно для:

  • отображения маршруток на карте в реальном времени;
  • обеспечения работы сервиса отслеживания транспорта.

3. Передача данных третьим лицам

Мы не передаём персональные данные пользователей третьим лицам. Данные используются только внутри сервиса Хатсайр.

4. Хранение данных

Данные о местоположении хранятся только в течение времени, необходимого для работы сервиса, и не используются для идентификации личности.

5. Безопасность

Мы принимаем разумные меры для защиты данных от несанкционированного доступа.

6. Изменения политики

Мы можем обновлять данную политику конфиденциальности. Актуальная версия всегда доступна на этой странице.
