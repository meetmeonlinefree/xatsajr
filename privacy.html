Политика конфиденциальности — Хатсайр Водитель

Приложение «Хатсайр Водитель» уважает конфиденциальность пользователей и собирает данные о местоположении устройства исключительно для работы сервиса отслеживания маршруток.

1. Сбор информации

Приложение собирает **координаты устройства** для передачи текущей позиции маршрутки.

2. Использование данных

  • отображение маршрутки на карте в приложении для пользователей;
  • обеспечение корректной работы сервиса отслеживания транспорта.

3. Передача данных третьим лицам

Данные не передаются сторонним лицам и используются только внутри сервиса Хатсайр.

4. Хранение данных

Данные хранятся только на время работы приложения и не используются для идентификации личности.

5. Безопасность

Приложение принимает разумные меры для защиты данных от несанкционированного доступа.

6. Изменения политики

Мы можем обновлять данную политику конфиденциальности. Актуальная версия всегда доступна на этой странице.
